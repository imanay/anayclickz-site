This website is built only with:
- HTML
- CSS
- JavaScript

Files:
- index.html
- styles.css
- script.js
- gallery images

How to use:
1. Create a new public GitHub repository.
2. Upload all extracted files into the root of the repo.
3. Go to Settings > Pages.
4. Set Source to 'Deploy from a branch'.
5. Choose branch 'main' and folder '/ (root)'.
6. Save and wait 1-3 minutes.

Your site will then be live.
