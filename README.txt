ANAY CLICKZ WEBSITE — CUSTOM DOMAIN READY

Files:
- index.html       -> main website
- sitemap.xml      -> SEO sitemap (replace yourdomain.com with your real domain)
- robots.txt       -> search engine instructions (replace yourdomain.com with your real domain)
- CNAME.example    -> rename to CNAME for GitHub Pages and replace with your real domain
- _redirects       -> helpful for Netlify routing

How to publish with a custom domain:

NETLIFY
1. Create a free Netlify account.
2. Drag and drop all files in this folder to Netlify.
3. In Netlify Site settings > Domain management, add your custom domain.
4. In your domain registrar (GoDaddy, Namecheap, etc.), point the domain to Netlify.
5. Replace "yourdomain.com" in sitemap.xml and robots.txt with your real domain.

GITHUB PAGES
1. Upload all files in this folder to a GitHub repository.
2. Rename CNAME.example to CNAME and replace the content with your real domain.
3. Enable GitHub Pages in the repository settings.
4. In your domain registrar, point the domain to GitHub Pages.
5. Replace "yourdomain.com" in sitemap.xml and robots.txt with your real domain.

Google indexing:
- After the site is live, submit https://yourdomain.com/sitemap.xml in Google Search Console.
- Google may index the site over time; it is not instant.

Important:
- Instagram embeds may not load everywhere; fallback links still work.
- For stronger Google ranking later, add your own photo text, captions, and an about page with more original wording.
